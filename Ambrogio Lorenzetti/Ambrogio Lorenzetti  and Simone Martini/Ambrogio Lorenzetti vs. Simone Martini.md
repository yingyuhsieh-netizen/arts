# Ambrogio Lorenzetti vs. Simone Martini：西恩納雙星的風格與思想比較

> 14世紀西恩納畫派的兩條路：一個仰望天國，一個俯視人間。

---

## 二、美學風格：優雅 vs 真實

### 1. 線條與色彩

**Simone Martini：宮廷的絲綢**
- 追求S型曲線，人物姿態如舞蹈般優雅
- 衣袍皺摺流暢如水，大量使用金箔與群青藍
- 風格如《Vogue》宗教版封面，高貴、裝飾性、脫俗
- 典型作品：《聖母領報》中天使加百列的衣角仍帶風勢，瑪利亞的驚懼被描繪得精緻而節制

**Ambrogio Lorenzetti：市民的麻布**
- 線條厚重，人物結實有體積感
- 色彩偏土色、灰色，但更貼近現實光影
- 風格如《國家地理》西恩納專刊，寫實、敘事、入世
- 典型作品：《好政府》裡跳舞的少女、蓋房的工人，每個人的衣著都有勞動的痕跡

### 2. 空間與透視

**Simone：神聖的金色背景**
- 繼承杜喬傳統，以金箔隔絕塵世時空
- 建築僅作為舞台布景，服務於人物
- 比例可為敘事犧牲，如《聖馬丁》中的馬
- 空間是象徵性的，目的在引導觀者仰望天國

**Ambrogio：理性的城市縱深**
- 喬托之後最激進的空間實驗者
- 《好政府》使用消失點透視雛形，畫出可「走進去」的街道
- 首次描繪雪景、雨景，追求捕捉自然瞬間
- 空間是經驗性的，目的是讓觀者審視自己生活的城市
- 比布魯內萊斯基的科學透視早近100年

### 3. 人物與情感

**Simone：神聖戲劇的主角**
- 情感高貴、內斂，如精心編排的舞台劇
- 聖人皆有王公貴族氣質
- 把拜占庭聖像的僵硬轉化為哥德式抒情

**Ambrogio：世俗社會的群像**
- 關心「群體」而非「個體聖人」
- 《好政府》繪製60+不同職業的市民，各有動態
- 聖母子更像現實中的母親與嬰兒
- 開啟了繪畫中的人文主義關懷

## 三、題材與思想：為誰而畫？

### 1. Simone Martini：服務於普世教會與君權

1. **國際巨星**：西恩納畫派第一位走出義大利的畫家，活躍於那不勒斯宮廷與亞維儂教廷
2. **題材核心**：聖母、聖人、國王加冕，讚頌神權與君權合一的封建秩序
3. **功能**：掛在禮拜堂、宮廷中，供貴族與主教默觀
4. **影響**：將西恩納的優雅帶向法國，成為國際哥德式風格的源頭之一

### 2. Ambrogio Lorenzetti：服務於城市共和國

1. **公民畫家**：主戰場是西恩納市政廳「九人執政團」會議室
2. **題材革命**：《好政府與壞政府的寓言》是歐洲第一幅完整的世俗政治宣傳壁畫
3. **功能**：教育市民，闡釋「選賢與能」的後果。好政府下商業繁榮、司法公正；壞政府下只有暴政與恐懼
4. **思想**：首次將「正義」「和平」「暴政」等抽象概念擬人化，繪畫成為公民道德課本
5. **影響**：開啟了「政治」作為繪畫主題的傳統，是公民人文主義的視覺宣言

**一句話差別**：Simone往天上看，為教宗與國王服務；Ambrogio往人間看，為市政府與市民服務。

## 四、技術與後世影響

### 1. 技術特徵

|  | Simone Martini | Ambrogio Lorenzetti |
| --- | --- | --- |
| **擅長媒材** | 蛋彩木板畫、金箔工藝、細密畫 | 大型濕壁畫 |
| **技術亮點** | 金箔壓花、鏤刻，創造華麗肌理 | 開發新灰泥配方，解決大面積壁畫保存問題 |
| **畫面追求** | 極致細節與裝飾性 | 整體敘事與空間效果 |

### 2. 歷史影響路徑

**Simone的線條 → 向北傳播**
- 影響法國、波西米亞的宮廷藝術
- 形成「國際哥德式」風格，影響林堡兄弟、早期揚·凡·艾克
- 文藝復興後被視為「優美但過時」的舊風格

**Ambrogio的空間 → 向南滲透**
- 透視實驗啟發佛羅倫斯的馬薩喬
- 世俗群像傳統影響後來的布魯哲爾
- 城市與政治觀念影響了文藝復興的政治哲學
- 瓦薩里承認他是喬托之後最重要的人物

## 五、結論：西恩納拋出的硬幣

Ambrogio Lorenzetti 和 Simone Martini 是14世紀西恩納拋出的一枚硬幣的兩面。

**Simone是正面**。他用盡中世紀所有的黃金、群青與優雅線條，為神權與君權時代譜寫最後的輓歌。他證明了哥德式藝術能達到何等抒情、精緻的高度。

**Ambrogio是反面**。他彎下腰描繪磚瓦、塵土與跳舞的市民，用壁畫寫下第一本公民課本。他預告了文藝復興的核心轉向：藝術不再只仰望神，也開始凝視人，凝視人所創造的城市與社會。

1348年黑死病席捲西恩納，Ambrogio極可能死於此疫，西恩納畫派就此衰落。四年後Simone逝於亞維儂。兩人的死亡，標誌著一個時代的終結。

他們留下的問題至今仍在：**藝術，究竟該用來裝飾權力，還是用來凝視生活？** 

Simone與Ambrogio用一生給出了兩種答案。兩種都極美，但只有一種推開了通往未來的大門。
